Control Flow¶

Control flow allows different code to execute depending on a condition. The keywords: if, elif and else are used to control execution of code.

Run the following code in the editor to see the output:

condition = True

if condition:
    print("The condition is", condition)

You should see The condition is True print. If we add the else keyword, we can have our code perform another action in the event that our test condition is false:

condition = False

if condition:
    print("The condition is", condition)
else:
    print("The condition must be false!!!")

You should see The condition must be false!!! print. If we add the elif keyword, we can test another condition:

condition = False
other_condition = True

if condition:
    print("The condition is", condition)
elif other_condition:
    print("The other condition is", other_condition)
else:
    print("Both conditions must be false!!!")

You should see The other condition is True print. What would print if both conditions are false?
